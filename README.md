# MalayMaster
MalayMaster is a web-based application designed to help users improve their Malay language skills through interactive learning, quizzes, and AI-powered assistance. It is built to make learning Malay easier, more engaging, and accessible for everyone.

✨ Features
📚 Interactive Malay learning modules
🧠 Practice quizzes and exercises
🤖 AI chatbot assistance (translation & explanations)
📊 Progress tracking for users
🎯 Simple and user-friendly interface
